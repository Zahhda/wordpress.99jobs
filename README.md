# 99Jobs

99Jobs is a Vite + React + TypeScript web app for job-seeker landing and conversion flows.
The visual direction is aligned to a clean, content-first style inspired by WordPress Twenty Twenty-Two.

## Tech stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui

## Local development

```sh
npm install
npm run dev
```

## Build and preview

```sh
npm run build
npm run preview
```

## Test

```sh
npm run test
```
